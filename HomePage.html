Selamat Datang!

Selamat datang di web pembelajaran ini. Pertama-tama, web pembelajaran ini terdiri dari 3 halaman, yakni:

-> Home page: Informasi/alat pandu terkait pengggunaan website, layout halaman dan sebagai "tutorial" bagi pengguna untuk menelusuri webpage ini.

-> Archives: Kumpulan catatan dan foto dari beberapa mapel, seperti Fisika, Matematika dan Ekonomi, ditambah dengan fitur "sidebox" disamping yang bisa digunakan dengan menggunakan tanda panah di bagian kiri screen, sidebox ini dapat digunakan untuk meloncat ke bab-bab pada page tersebut.

-> About Me: Sekedar halaman untuk lebih mengenal saya, Yohanes Christopher Hardjito sebagai developer dari website ini.


Maka dari hal tersebut, saya harap bahwa situs web ini setidaknya dapat berguna bagi anda, jika terdapat kendala maka dipersilahkan untuk kembali ke homepage, dan review tutorial ini lain waktu. Untuk ke page/slide lain bisa click salah satu dari 3 tombol yang berada pada navigation bar halaman ini.
